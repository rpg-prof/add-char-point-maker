Personagens com essa perícia são excepcionais atiradores com arcos e bestas. Eles têm uma visão excepcional para distâncias, bom senso parar medir um tiro longo e paciência para recarregar e disparar com sua arma. Se o personagem optar por levar uma rodada inteira para dar apenas um tiro, ele não sofrerá penalidades para média distância, e apenas uma penalidade de -2 para longas distâncias. Mesmo que possa fazer ataques múltiplos, o personagem só ganhará o bônus se realizar apenas um ataque.
